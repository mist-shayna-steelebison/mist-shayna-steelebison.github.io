How to Pay Bills as a Bookkeeper

Go to E.R. Griffis Website

By Shayna SteeleBison


Step 1: Receive the mail from the mail carrier and process it.

When the mail carrier arrives in the afternoon, greet them and take custody of the mail. Throw away any mail that appears to be junk or solicitations.

Mail received will be regarding different property addresses. Sort the mail by property address.

Step 2: Open each Envelope.

It is recommended to use an electronic letter opener to open the mail in order to be the most efficient and save the most time. Make a stack of the open envelopes.

Step 3: Remove mail from each envelope.

Remove the mail from each envelope and paperclip the sheets of paper together with the return envelopes. If a bill does not have a return envelope than paperclip a blank envelope to the bill.

Step 4: Write the name of the property the bill is for.

For each bill in the pile please write the property name on the bill so that you can be sure to pay it from the correct checkbook.

Mail received will be regarding different property addresses. Sort the mail by property address.

Step 5: Pay the bills.
  1. Write checks to pay each bill
  2. place checks in the remittance envelope
  3. place a return address and a stamp on the envelope
